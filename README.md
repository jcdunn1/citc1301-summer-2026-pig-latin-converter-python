Pig Latin is a word game (usually used by kids to "try" and fool adults) that changes English words using these rules:

### Rule 1 — Word Begins with a Vowel
If a word begins with a vowel (`a`, `e`, `i`, `o`, `u`):

- Add `"yay"` to the end of the word.

#### Example

| Original | Pig Latin |
|---|---|
| apple | appleyay |
| orange | orangeyay |

### Rule 2 — Word Begins with a Consonant
If a word begins with a consonant:

- Move the first letter to the end of the word
- Add `"ay"`

#### Example

| Original | Pig Latin |
|---|---|
| dog | ogday |
| cat | atcay |

### Case Handling

Your program should convert the word to lowercase before converting it to Pig Latin. Therefore, all Pig Latin output should be lowercase.

For example, `Smile` becomes `milesay`, not `MileSay`.
